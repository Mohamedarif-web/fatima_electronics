
# Fatima Electronics Icon Creation

## Quick Setup:

1. **Open the icon generator:**
   - Open `create_icons.html` in your web browser
   - You'll see a green FE icon with 'ELECTRONICS' text

2. **Download required sizes:**
   - Click 'Download 256x256' and save as `icon-256.png`
   - Click 'Download 512x512' and save as `icon-512.png`

3. **Convert for different platforms:**
   
   **Windows (.ico):**
   - Go to: https://convertio.co/png-ico/
   - Upload `icon-256.png`
   - Download as `icon.ico`
   - Place in `build/icon.ico`
   
   **macOS (.icns):**
   - Go to: https://convertio.co/png-icns/
   - Upload `icon-512.png`
   - Download as `icon.icns`
   - Place in `build/icon.icns`
   
   **Linux (.png):**
   - Rename `icon-512.png` to `icon.png`
   - Place in `build/icon.png`

## Your app will now show:
- ✅ Application Name: 'Fatima Electronics'
- ✅ Window Title: 'Fatima Electronics' 
- ✅ Custom FE icon in taskbar/dock
- ✅ Green theme colors in titlebar (macOS)
